# SRS (Super Rotation System) 技术报告

## 1. SRS 概述

SRS 是现代俄罗斯方块的标准旋转系统，核心特性：

1. **纯旋转** - 方块绕固定点旋转，不结合位移
2. **墙踢 (Wall Kick)** - 旋转被阻挡时，自动尝试5个备用位置
3. **不同方块不同策略** - J/L/S/T/Z 共享kick表，I独立，O不旋转

## 2. 生成方向 (Spawn Orientation)

| 方块 | 生成状态 | 居中方式 |
|------|----------|----------|
| I | 水平 (0°) | 居中 (与网格线交叉) |
| O | 水平 (0°) | 居中 (与网格线交叉) |
| J/L/T | 平坦朝左 (0°) | 偏左 |
| S/Z | 水平 (0°) | 偏左 |

旋转状态定义：
- **0**: 生成态
- **R**: 顺时针旋转一次
- **2**: 旋转180°
- **L**: 逆时针旋转一次

## 3. 旋转中心

| 方块 | 旋转中心 |
|------|----------|
| J/L/S/T/Z | 某块中心 |
| I | 网格线交点 |
| O | 块中心 (但O不旋转) |

## 5. 实现策略

1. **存储旋转状态** - 每个方块记录当前 rotation: 0/R/2/L
2. **定义生成态矩阵** - 每个方块的4个旋转状态矩阵
3. **Kick表查询** - 根据当前状态→目标状态查表
4. **尝试5次Kick** - 依次尝试，失败则回退尝试下一个

# Tetris HTML - 现代方块游戏规格

## 项目概述

- **项目名称**: Tetris HTML
- **类型**: HTML5 网页游戏
- **核心功能**: 经典俄罗斯方块现代版，支持 SRS 旋转、Hold、幽灵方块等现代机制

## 玩法说明

### 操作键位

| 按键 | 功能 |
|------|------|
| ← / → | 左右移动 |
| ↓ | 软下落（加速） |
| ↑ / X | 顺时针旋转 |
| Z | 逆时针旋转 |
| Space | 硬drop（直接落底） |
| C / Shift | Hold（暂存方块） |

### 计分规则

- 1行: 100
- 2行: 300
- 3行: 500
- 4行 (Tetris): 800
- T-Spin: 根据消行数额外加分
- Back-to-back: 连续 Tetris 或 T-Spin 乘1.5

## 技术规格

### 网格

- 标准 10列 x 20行
- 方块大小: 30px

### SRS 旋转系统

- 7种方块: I, O, T, S, Z, J, L
- 旋转规则使用标准 SRS 表
- 支持墙踢 (Wall Kick)

### 方块颜色

| 方块 | 颜色 |
|------|------|
| I | 青色 |
| O | 黄色 |
| T | 紫色 |
| S | 绿色 |
| Z | 红色 |
| J | 蓝色 |
| L | 橙色 |
